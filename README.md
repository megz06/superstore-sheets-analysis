# Superstore Sales Analysis (Google Sheets)

## Overview
Analysis of the Sample Superstore dataset (~9,994 orders) using Google Sheets,
covering sales, profit, regional, and category performance.

- Total Sales: $286397.0217
- Total Profit: $286397.0217
- Highest performing region: Central ($501239.8908)
- Most profitable category: [Technology]
- Top-selling product: [Canon image GLASS 2200 Advanced Copier]

- Loss-making orders: [1871] out of 9,994
